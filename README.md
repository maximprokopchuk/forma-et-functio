# Forma et Functio

Образовательный портал по веб-дизайну.

**Название** — отсылка к принципу Луиса Салливана "form follows function" (1896), фундаментальному для индустриального и цифрового дизайна.

## Документация

- [План имплементации](docs/implementation-plan.md)

## Статус

В разработке. См. фазы имплементации в плане.

## Референс

Архитектура повторяет проект [diabolus-in-musica](https://github.com/maximprokopchuk/diabolus-in-musica) — образовательный портал по теории музыки.
